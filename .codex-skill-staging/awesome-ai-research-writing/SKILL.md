# Awesome AI Research Writing

This skill packages the prompt library from the `Leey21/awesome-ai-research-writing` repository into a Codex skill.

Use this skill when the user asks for:
- academic writing help
- Chinese/English paper translation
- polishing or rewriting paper text
- logic checks or reviewer-style feedback on a draft
- experiment analysis
- figure or table title generation

## Workflow

1. Open `README.md` in this skill directory.
2. Identify the section that best matches the user's request.
3. Apply the repository's prompt intent and constraints to produce the answer directly.
4. Prefer concise, publication-oriented wording and preserve technical meaning.

## Guidance

- Treat the repository as a prompt and workflow reference, not as something to quote verbatim at length.
- If the user provides draft text, operate on that text directly.
- If multiple sections are relevant, combine them pragmatically instead of forcing a single template.
- When working with LaTeX content, preserve formulas and escape special characters only when needed by the requested output format.

## Files

- `README.md`: upstream prompt collection and usage notes
- `images/`: repository images retained for reference
